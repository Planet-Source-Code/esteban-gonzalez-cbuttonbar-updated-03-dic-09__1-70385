Description:

cButtonBar is a class that uses a picturebox control as a container to create a bar of buttons with some especial features like text alignment, use of icons, use of skins, among others.
This button bar can be used as a toolbar thought it allows the use of buttons and separators; it also can be used as a group of tabs because it allows a button to keep its down state; or simply as a group of buttons.

Some of the features of this class are:

 Different size buttons.
 Text and icon alignment.
 General skins (all or some of the buttons can use the same skeleton skin).
 Particular skins (an image that represents the five states of the buttons).
 Windows 98 look like buttons.
 Auto adjust button size.
 MouseEnter, MouseLeave events.
 Buttons effects (change text color, background color and border color depending on the state of the button - Normal, Focused, Over, Down and Disabled -).
