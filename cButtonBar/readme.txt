Descripción:
cButtonBar es una clase que utiliza un control picturebox para crear una barra de botones que poseen algunas características especiales como texto alineado horizontal y verticalmente, utilización de iconos en los botones, uso de skins para los botones, entre otras.
Esta barra de botones puede utilizarse como una barra de herramientas ya que permite el uso de botones y separadores, también puede utilizarse como una barra de pestañas (Tabs) ya que permite que un botón se mantenga en el estado presionado, o simplemente como un conjunto de botones comunes.

Algunas de las características y futuras modificaciones de esta clase son:

· Permite crear botones de distinto tamaño.
· Permite alinear los iconos y el texto.
· Permite el uso de una skin general (Todos los botones tienen el mismo estilo).
· Permite el uso de una skin particular (Cada botón tiene una skin propia).
· Los botones pueden tener el aspecto de Windows 98.
· Los botones creados pueden ajustar su tamaño al contenido del mismo.
· Los botones poseen algunas características más que los botones comunes de Visual Basic, como:
   + Eventos MouseEnter, MouseLeave
   + Permite ciertos efectos en el botón (cambio de color de texto, de fondo, de los bordes de acuerdo al estado del mismo - Normal, Con Foco, Con Captura del Mouse, Presionado y Deshabilitado-).

IMPORTANTE: Algunas características todavía no están completamente implementadas.
